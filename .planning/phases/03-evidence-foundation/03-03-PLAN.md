**Create src/app/dashboard/evidence/actions.ts** (Server Actions):

Mark with 'use server' at top.

1. `createEvidenceAction(formData: FormData)`:
   - Get user via requireUser()
   - Extract and validate fields from FormData using EvidenceItemCreateSchema.safeParse()
   - If validation fails, return { error: formatted error message }
   - Call createEvidenceItem() with validated data + userId + isManual: true + confidence: 1.0
   - revalidatePath('/dashboard/evidence')
   - redirect to /dashboard/evidence

2. `updateEvidenceAction(formData: FormData)`:
   - Get user via requireUser()
   - Extract item ID from FormData
   - Extract and validate fields using EvidenceItemUpdateSchema.safeParse()
   - If validation fails, return { error: formatted error message }
   - Call updateEvidenceItem(id, userId, validatedData)
   - revalidatePath('/dashboard/evidence')
   - redirect to /dashboard/evidence

3. `deleteEvidenceAction(formData: FormData)`:
   - Get user via requireUser()
   - Extract item ID from FormData
   - Call deleteEvidenceItem(id, userId)
   - revalidatePath('/dashboard/evidence')
   - Return (no redirect, page revalidates in-place)

Import revalidatePath from 'next/cache', redirect from 'next/navigation'.
