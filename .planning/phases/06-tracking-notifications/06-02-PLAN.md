  <action>
    Create src/app/api/roles/[jobId]/status/route.ts with two handlers:

    **PATCH handler:**
    1. Call verifySession() per DEV-024 (import from '@/lib/auth/session'). Return 401 JSON if no session.
    2. Extract jobId from params (Next.js App Router: `{ params }: { params: Promise<{ jobId: string }> }`, await params)
    3. Parse request body with UpdateRoleStatusSchema.parse(). Catch ZodError, return 400 with validation errors.
    4. Fetch current status BEFORE update (for audit before_value): getRoleStatusForJob(userId, jobId)
    5. Call upsertRoleStatus(userId, jobId, validated.status, validated.notes)
    6. Log audit trail: logParserAudit({
         userId: session.id,
         entityType: 'role_status',
         action: 'status_change',
         entityId: jobId,
         beforeValue: previousStatus ? { status: previousStatus.status, notes: previousStatus.notes } : null,
         afterValue: { status: validated.status, notes: validated.notes },
         source: 'user'
       })
    7. Return Response.json({ success: true, status: validated.status })

    **GET handler:**
    1. Call verifySession(). Return 401 if no session.
    2. Extract jobId from params.
    3. Call getRoleStatusForJob(userId, jobId).
    4. Return Response.json({ status: result?.status ?? null, notes: result?.notes ?? null })

    Error handling: try/catch around each handler, return 500 with { error: 'Internal server error' } on unexpected errors.

  </action>

  </verify>
